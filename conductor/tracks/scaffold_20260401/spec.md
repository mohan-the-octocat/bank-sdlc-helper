# Specification - Scaffold Extension and Define Core Skill Structure

## Goal
Initialize the project as a Gemini CLI Extension and establish the modular structure for team-specific skills.

## Requirements
- Create `package.json` with correct extension metadata.
- Create `GEMINI.md` for project-level context.
- Define the `skills/` directory structure.
- Initialize individual team skill folders (Product, Design, BSG, Architecture, Engineering, QA).
- Add a placeholder `SKILL.md` to each team folder.
