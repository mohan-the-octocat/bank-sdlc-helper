## Bank SDLC Workflow

| Team | Function |
| :---- | :---- |
| Product Team | Capture all regulatory requirements as Checklist Capability to do competitive research and suggest Come up with a BRD solution |
| Design Team | Generate the Design Compare the design against competition |
| BSG - Business Solution Group - Business Analyst (Techno Functional guys) | Functional Specification Documents Non-Functional Specification Document User Stories - Use cases Own the UAT - Functional Testing/ In-sprint testing |
| Solution Architecture | Chooses the right tools for fulfilling the requirements Drafts the Solution Architecture based on best practices, guidelines & Bank’s policies |
| Infra/ Tech Architecture | Maps the tools shortlisted for the job to the Enterprise Preferred tools |
| Engineering Lead | Builds the low-level Spec Language nuances |
| Engineering team | Builds the actual code |
| Functional Testing | Build the Functional test cases required |
| Release Engineering | Generate the Release notes Perform the Deployment sequence Tag the right builds Deploy to the right deployments |
