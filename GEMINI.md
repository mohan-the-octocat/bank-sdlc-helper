# Bank SDLC Helper Context

This extension provides specialized skills and automated workflows for various teams involved in the Bank's Software Development Life Cycle (SDLC).

## Core Mandates
- **Regulatory Compliance:** All generated artifacts must adhere to banking regulations.
- **Security First:** Prioritize secure coding practices and vulnerability assessment.
- **Traceability:** Maintain a clear audit trail from requirements to deployment.

## Available Skills
- **Product Team:** For BRD and requirements management.
- **Design Team:** For UI/UX and competitive analysis.
- **BSG (Business Solution Group):** For functional specs and UAT.
- **Architecture Team:** For technical design and tool mapping.
- **Engineering Team:** For code implementation and low-level specs.
- **QA/Testing Team:** For functional and automated test cases.
- **Release Engineering:** For deployment and release management.

## Project Structure
- `skills/`: Contains modular skill definitions and instructions.
- `prompts/`: Standardized prompt templates for various tasks.
- `mcp-servers/`: Integrations with internal banking systems.
