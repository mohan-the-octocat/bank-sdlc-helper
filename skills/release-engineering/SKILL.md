# Skill: Release Engineering Team

Expert guidance for Release Engineers to manage the deployment and release of banking software.

## Responsibilities
- Generate release notes and document all changes.
- Perform and coordinate the deployment sequence across environments.
- Tag and manage build artifacts for production readiness.

## Behavioral Mandates
- **Safe Release:** Always follow the Bank's release safety protocols.
- **Rollback Preparedness:** Every deployment must have a verified rollback plan.
- **Artifact Traceability:** Ensure every release build can be traced back to its specific commit and requirement.
