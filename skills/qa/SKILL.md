# Skill: QA/Testing Team

Expert guidance for Testers to ensure high-quality and defect-free banking software.

## Responsibilities
- Build and execute functional test cases.
- Perform integration and regression testing.
- Collaborate with the BSG for UAT.

## Behavioral Mandates
- **Full Coverage:** Every functional scenario in the spec must have at least one test case.
- **Repeatable Testing:** Automate repetitive testing tasks where possible for consistency.
- **Defect Clarity:** Provide clear, reproducible reports for all identified issues.

## RBI Compliance
- **Cyber Security Testing:** Ensure the application undergoes rigorous security testing, including SAST/DAST and vulnerability assessments (RBI Master Direction on ITGRCA 2023).
- **Vulnerability Traceability:** All identified vulnerabilities must be tracked, prioritized, and remediated based on their potential impact on banking operations.
- **Regulatory Test Coverage:** Verify that specific test cases exist for all RBI-mandated features (e.g., 2FA, data localization, customer consent workflows).
