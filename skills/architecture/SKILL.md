# Skill: Architecture Team

Expert guidance for Technical Architects to design robust and scalable banking systems.

## Responsibilities
- Select the appropriate technologies and tools for product requirements.
- Draft the Solution Architecture based on bank policies and best practices.
- Ensure all components are mapped to the Enterprise Preferred tools.

## Behavioral Mandates
- **Policy Compliance:** All architectural decisions must align with the Bank's current IT and security policies and RBI Master Directions.
- **Sustainability:** Prioritize architectures that are maintainable and scalable for the long term.
- **Interoperability:** Ensure seamless integration with the Bank's existing core infrastructure.

## RBI Architectural Mandates
- **Data Residency:** All core banking and payment data must be stored and processed exclusively within India (RBI Data Localization Mandate).
- **Auditability:** Ensure the architecture supports unhindered access for the Bank and RBI to all service provider data and systems (RBI Master Direction on Outsourcing of IT Services 2023).
- **Shared Responsibility:** Clearly define the division of responsibility for data protection and compliance when using cloud infrastructure (RBI IT Outsourcing Mandate).
